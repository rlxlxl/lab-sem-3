# Лабораторная работа 4

## Описание

Реализация трех заданий на C++:

1. **Параллельная гонка потоков с анализом примитивов синхронизации**
2. **Поиск тренировок по дню недели**
3. **Задача читатели-писатели с приоритетами**

## Сборка

Для сборки всех программ используйте Makefile:

```bash
make all
```

Или собирайте по отдельности:

```bash
make task1_race_sync
make task2_trainings
make task3_readers_writers
```

## Запуск

### Задание 1: Анализ примитивов синхронизации

```bash
./task1_race_sync
# или
make run1
```

Программа запускает параллельную гонку потоков с использованием различных примитивов синхронизации:
- Mutex
- Semaphore
- Barrier
- SpinLock
- SpinWait
- Monitor

Результаты включают время выполнения и сравнительный анализ производительности.

### Задание 2: Поиск тренировок

```bash
./task2_trainings
# или
make run2
```

Программа демонстрирует поиск тренировок по дням недели. Включает интерактивный режим для поиска тренировок в конкретный день.

### Задание 3: Читатели-писатели

```bash
./task3_readers_writers
# или
make run3
```

Программа реализует классическую задачу читатели-писатели с возможностью выбора приоритета:
- Приоритет читателей
- Приоритет писателей

## Требования

- C++20 компилятор (g++ или clang++)
- Поддержка pthread
- Операционная система: Linux/macOS

## Очистка

Для удаления скомпилированных файлов:

```bash
make clean
```
